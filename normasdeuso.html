Normas de Uso:

Uso de la membresía del parqueadero

La cuenta del usuario (aprendiz, instructores, personal administrativo y personal logístico) solo es para registrar y dar uso de parqueo a los miembros antes mencionados e invitados autorizado, dicho esto la cuenta no debe ser usada para registrar o hacer uso del parqueadero mediante el sistema linecuping a personas particulares.

La membresía permite registrar un máximo de tres vehículos (bicicleta, Moto, Automóvil) por usuario, pero solo permite el uso del sistema para parquear uno de los tres.

Penalizaciones por uso indebido.

Si el usuario, no hace un uso correcto o no respeta la normas establecidas para el uso del parqueadero, el sistema Linecuping generara un estado del usuario indicando que esta bajo penalización, cuya penalización es el restringir el uso del parqueadero. Dicho tiempo de restricción queda bajo la determinación del coordinador de sede.

Las normas a respetar son las siguientes:

No parquear en las vías de trafico dentro del parqueadero, las cuales están señalizadas.

Hacer ingreso del vehículo (bicicleta, moto, automóvil) sin hacer el debido proceso de registro en el sistema Linecuping.

Ocupar sectores no determinados en el parqueadero para el vehículo (bicicleta, moto, automóvil) por ejemplo, parquear una moto en el sector de las bicicletas sin ser autorizado.

Responsabilidades del SENA.

El SENA, no se hace responsable por perdidas de vehículos u objetos, relacionados con el usuario; debido a que el servicio de parqueadero se hace sin animo de lucro. No diciendo con esto que el servicio esta para prestarse con la mayor eficiencia y responsabilidad que se pueda brindar.
